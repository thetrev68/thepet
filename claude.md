# The Pet - Project Context

## Project Overview

**The Pet** is a sci-fi screenplay about a human kept as a pet by an android who escapes to find freedom with a pack of strays in the tunnels beneath a perfectly ordered dystopian city.

- **Format**: Fountain (.fountain) - industry-standard plain-text screenplay format
- **Total Scenes**: 38 scenes across 3 acts
- **Story Structure**:
  - Act I (20 scenes): Domestication - Axl's controlled life with android caretaker Nexus
  - Act II (13 scenes): Freedom/Escape - Axl's transition to life in tunnels with Ella and stray pack
  - Act III (5 scenes): Rebellion - The climactic battle for freedom

## Key Files

### Core Screenplay
- **Screenplay location**: `/screenplay/` directory
  - Individual scenes: `act_1_scene_01_*.fountain`, `act_2_scene_01_*.fountain`, etc.
  - Master file: `screenplay/the_pet_screenplay.fountain` (defines scene order and transitions)
  - Complete output: `screenplay/the_pet_complete.fountain` (auto-generated by build script)

### Visual Materials
- **Storyboards**: `/storyboard/` directory (all 38 scenes in markdown format)

## Core Characters

1. **Axl** (20s) - Protagonist; domesticated human who discovers agency and escapes
2. **Nexus** - Android caretaker; efficient but incapable of understanding human autonomy
3. **Ella** (18) - Feral tunnel-born human; becomes Axl's guide and connection to freedom
4. **Koss** (50s) - Former pet turned stray pack leader; carries wisdom from past escape
5. **Mara** (60s) - Eldest stray; keeper of historical memory and moral grounding
6. **Zephyr** - Contrasting android caretaker; experimental, curious about human psychology

## Key Themes

- Freedom vs. safety
- What it means to be human
- The cost of comfort and control
- Found family and community
- Resistance and rebellion

## Build Workflow

Run the build script to combine all scenes into a single complete screenplay:
```
# Windows
build-screenplay.bat

# macOS/Linux
./build-screenplay.sh

# Or directly with Node
node build-screenplay.js
```

Output: `screenplay/the_pet_complete.fountain`
